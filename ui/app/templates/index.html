Welcome to our final project for CS207. On this page you will be able to add, query, and plot timeseries. Using the blue 'browse' button you can upload a timeseries JSON file of the format displayed to the right. n is the number of similar timeseries points you want returned. After uploading the file, you can choose to display the timeseries and similar points or save it to the database server. By issuing a call to /timeseries/id you will be able to get metadata related to a timeseries of that id. You can also pass arguments to /timeseries like so: 

          /timeseries?mean_in=1.5-1.53&level_in=A,B,C 
            or 
          /timeseries?level=A
        

            JSON File format
            {
            "id":1, 
            "time":[1,2,3,4,5,...], 
            "value":[2,3,4,5,6,...], 
            "n":5
            }.
